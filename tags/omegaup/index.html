Categorías omegaUp

Conoce las Categorías de Usuarios de omegaUp Hemos asignando una categoría a cada usuario de omegaUp. La categoría de cada usuario depende su posición actual en el ranking de omegaUp como se muestra a continuación: El top 1% de usuarios mejor posicionados en el rank obtiene la categoría ω (“omega pequeña”) y color rojo. El top 9% de usuarios siguientes obtiene categoría Ω (“omega grande”) y color amarillo. El top 15% de usuarios siguientes obtiene categoría Θ (“zeta” o “theta”) y color azul.

Anunciando omegaUp Mentores

Hackathon omegaUp febrero 2017 - Resultados

En omegaUp organizamos nuestro primer Hackathon del año abierto al público, del 23 al 25 de Febrero pasados.
